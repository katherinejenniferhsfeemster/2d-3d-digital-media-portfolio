Selected cases

Six cases, one per tool. Each emits a real native project file — a Script-Fu `.scm`, an Inkscape CLI script, a Krita Python plugin, a Libresprite Lua script, a bpy render script, and a full Godot 4 project — plus the deterministic raster output so the whole portfolio renders without any tool installed.

Krita 5.2 · Python plugin · .kra

Painted scene layer exporter

A Krita Python plugin that walks the paint graph, hides every layer but one in turn, and exports per-layer masks + a COCO-style labels.json. Twelve seeded scenes ship as real .kra archives so the artefact opens in Krita unchanged.

Code you can actually run

Every artefact on this page is regenerated by GitHub Actions on every push — nothing is pre-baked. Clone the repo, run one script, and get identical tiles, icons, scenes, atlases, 3D frames and a Godot viewer.

GIMP — batch segmentation dataset

python3 src/build_gimp_case.py

Writes 64 tiles + masks, manifest.json, and the batch.scm Script-Fu file that GIMP would run in headless mode.

Inkscape — pictogram set

python3 src/build_inkscape_case.py

Emits 24 parametric SVGs, a 2× PNG contact sheet, and render.sh that shells out to inkscape --actions.

Krita — painted scenes + plugin

python3 src/build_krita_case.py

Writes 12 real .kra archives, merged renders, L-mode masks, per-scene COCO labels, and the Krita Python plugin source.

Libresprite — creature atlas

python3 src/build_libresprite_case.py

Packs 128 frames into a single atlas, writes Aseprite-compatible atlas.json with frameTags, and saves the Lua CLI script.

Blender — headless dataset

python3 src/build_blender_case.py

Produces 48 frames + masks + coco.json (960 annotations) and ships render_dataset.py, the real bpy entrypoint.

Godot — unified asset viewer

python3 src/build_godot_case.py

Generates a full Godot 4 project under cases/godot/viewer/ — project.godot, main.tscn, main.gd — that loads every PNG from the other five cases.

Stack

CLI actions API (--actions=export-type:png), parametric SVG generation, 2× DPI rasterization, palette-locked pictograms for product and research UIs.

Blender 4.x

bpy Python library, Cycles rendering, procedural scene graphs, world_to_camera_view for 2D/3D annotation, headless CI pipelines.

Godot 4 & CI

GDScript, DirAccess asset loaders, --headless runtime, GitHub Actions regenerating every artefact on push. Pillow + cairosvg as the universal rasterizer.
